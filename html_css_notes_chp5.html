HTML & CSS: Design and build websites

NOTES - Chapter 5: IMAGE

Objectives

This chapter covers:

  1. Add images to your html website
  2. Pick image format to use
  3. Show the image at the rigth size
  4. Optimize an image for loading speed of a website

Introduction

"A picture can say a thousand words"

Choose images:

  • Use images to set the tone for the website
  • Use your own photos.
    • If your don't have one, use stock images.
      • Stock Images: Images you pay to use(from a website).
      • Refer to pg. 97 for list of website to buy stock images.
      • All images are subject to copyright.
  • If you have several images, put them in simple, consistent background.
  • Images should:
    1. Be relevant
    2. Convey information
    3. Convey the mood
    4. Be recognisable
    5. Fit the color pallete
  • Online gallery from the book (refer to pg.97)

Storring images:

  • Create folders for images used on website.
    • Create sub-folder for different types of images
      • ex: logo, gif, background, foreground, etc...

Adding images:

  • Use <img> to add images.
    • Empty element (no closing tag)
    • Must carry 2 main attributes:
      1. src - tell the location of the image (path to get to the image)
        2. ex: src = "images/name.jpg"
      2. alt - provide text description of image when you can't see it (should give an accurate description of the image for those that are visually impared; use with screen reader).
        3. If image is used for aesthetics, alt will still be used but the quotes is empty
        ex: alt = "test image"
  • title - Provide additional information about the image
    • Most browser display the text in tooltip when hover mouse over the image.
    ex: title= "testing title for image"
  • height - Height of the image in pixels
  • width - width fo the image in pixels
  • Images take longer to load than html codes.
    • It's a good practice to specify the size of the image.
      • This helps the browser to render the page and leave a little space for the image to load (loading speed optimization).
      Common practice: Size of image is specified with CSS.
  • Example with all attributes mentioned above:
    • <img src= "images/name.jpg" alt= "test image" title= "this is a test only; nothing special" height= "100" width= "100">

Location of image:

The location of image rag in the code will affect how it is display:
There are 3 ways for the image to be displayed:
  1. Before paragraph
  2. Inside the start of a paragraph
